# 🕵️ CrimeLens

An interactive detective mystery game built using **HTML, CSS, JavaScript and JSON**.

---

## Features

- 🎬 Cinematic landing pagegit
- 🕵 Detective story
- 🗺 Multiple investigation locations
- 👥 Suspect profiles
- 🔍 Evidence collection
- ⏳ Countdown timer
- 🌙 Dark mode
- 🏆 Achievement system
- 💾 Local Storage
- 📱 Responsive design

---

## Technologies

- HTML5
- CSS3
- JavaScript
- JSON

---
